Три оси - hue, saturation и lightness

Hue

Тон цвета, представленный в виде угла цветового круга.

Saturation

Насыщенность цвета

Lightness

«Светлота» цвета

HSL

/*
* @param {<number> | <angle>} --hue Оттенок

* @param {<percentage>} --lightness Яркость
*/

JSON — это просто и удобно

  • Старый формат, поэтому хорошо импортить в другие языки
  • Очевидный синтаксис: могут писать все (дизайнеры и разработчики со всех платформ)

Какие линтеры добавили для репозитория

Палитру и схему иногда правят люди, не привыкшие писать код, поэтому:

  • Проверка синтаксиса json
  • Проверка синтаксиса hex
  • Нельзя использовать в схемах цвета, которых нет в палитре

Типизация

Мы типизировали переменные из палитры.

Зачем?...

            #el {
              background-color: red;
              animation: anim infinite 3s;
            }
            @keyframes anim {
              50% { background-color: blue; }
            }
        
            #el {
              --color-stop: red;
              background: var(--color-stop);
              animation: anim infinite 3s;
            }
            @keyframes anim {
              50% { --color-stop: blue; }
            }

Почему?

Потому что изначально css переменные не типизированы

            div {
              --name: red;
              --name: 11px;
            

                  --name: calc(14deg * 3);
                }
            

        
            #el {
              --color-stop: red;
              background: var(--color-stop);
              animation: anim infinite 3s;
            }
            @keyframes anim {
              50% { --color-stop: 10px; }
            }

Почему?

Браузер не знает, как построить переход от одного значения к другому

            red  =>  ???  =>  blue
            red  =>  ???  =>  10px

Типизация переменной

            CSS.registerProperty({
              name: '--color-stop',
              syntax: '<color>',
              inherits: false,
              initialValue: 'transparent'
            })
        
            #el {
              --color-stop: red;
              background: var(--color-stop);
              animation: anim infinite 3s;
            }
            @keyframes anim {
              50% { --color-stop: blue; }
            }

Поэтому типизируем

Object.keys(palette).forEach((colorName) => {
          js += `    CSS.registerProperty({
            name: '--${colorName}',
            syntax: '<color>',
            inherits: true,
            initialValue: '${palette[colorName]}',
          });\n`
        });

VDS (value definition syntax)

initial ключевые слова
<length> простые типы данных
<'line-height'> сложные типы данных (именуются обычно по свойству)
[<A> | <B>] <C> группировка
<A> || <B> || <C> один или более из группы в заданном порядке
<A> && <B> && <C> один или более из группы в любом порядке
<A>* ноль или более раз
<A>+ один или более раз
<A>? необязательный
<A> {n} А повторяется n раз
<A> {n, m} А повторяется не меньше n раз и не больше m раз
<A># А повторяется один или более раз, разделенных запятыми

Поэтому типизируем

export function registerColorVars() {
